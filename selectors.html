ID Selector

You can select the item on the webpage using its ID. In cypress function we pass the id name with hashtag (#) cy.get("#idname")

This is a dummy element with an id for example

Class Selector

You can select the item on the webpage using its class name. In cypress function we pass the class name with dot (.) cy.get(".classname")

This is a dummy element with the class name for example

Attribute Selector

You can select the item on the webpage using its attribute name. In cypress function we pass the attribute cy.get("data-color=["red"]")

This is a dummy element with the attribute name for example

Tag name Selector

You can select the item on the webpage using its attribute name. In cypress function we pass the attribute cy.get("data-color=["red"]")

We will select this item using its tag name without any id, class or attribute
